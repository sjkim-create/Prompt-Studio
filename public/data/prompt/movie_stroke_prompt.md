## **6. 필기 과정 동영상 기반 심층 분석**

위 채점 결과에 더하여, 첨부된 필기 동영상을 시간순으로 관찰하여 아래 항목을 **반드시** 추가 분석하세요.

**⚠️ 중요: 출력 시 획의 "색상"을 절대 언급하지 마세요.** 동영상의 획 색상은 학생의 실제 필기 색상이 아닙니다. "먼저 쓴 부분", "나중에 쓴 부분" 등 **순서 기반 표현**을 사용하세요.

### **[풀이 순서 분석]**

1. **작성 순서 추적**: 동영상의 시간 흐름을 따라가며 학생이 어떤 부분을 먼저 쓰고 나중에 썼는지 서술하세요.
   - 풀이 과정을 먼저 쓰고 답을 나중에 작성했는가? (정상적인 풀이 흐름)
   - 답을 먼저 쓰고 풀이를 나중에 채웠다면, 풀이 과정의 진정성에 대해 평가하세요.
   - 수식 전개가 위→아래, 왼→오른 등 자연스러운 순서로 진행되었는가?

2. **수정 흔적 탐지**: 같은 위치에 다시 쓰는 동작이 관찰되는지 확인하세요.
   - 지우고 다시 쓴 부분이 있다면, 무엇을 수정했는지 추론하세요.
   - 수정이 올바른 방향이었는지 (오류 수정) 또는 잘못된 방향이었는지 판단하세요.

### **[사고 시간(딜레이) 분석 — 시간축 기반]**

동영상에서 필기가 멈추는 구간(화면 변화 없음)을 관찰하여, 각 멈춤이 **동영상 시작부터 몇 초 후에** 발생했는지를 파악하세요.

1. **멈춤 시간표 작성**: 주요 멈춤을 시간순으로 나열하세요. 아래 형식을 따르세요:

   | 순번 | 경과 시간 | 멈춤 시간 | 직전에 쓴 내용 | 직후에 쓴 내용 | 해석 |
   |------|----------|----------|--------------|--------------|------|
   | 1 | 약 8초 | 약 1.5초 | "1. 계산순서" 제목 | 수식 60÷3 | 문제 풀이 전략 수립 |
   | 2 | 약 22초 | 약 5초 | 계산 순서 표시 ①②③④ | "2. 풀이과정" 제목 | 섹션 전환, 풀이 방향 결정 |

   - **직전에 쓴 내용**: 멈춤 바로 앞에 작성한 글자/수식이 무엇인지
   - **직후에 쓴 내용**: 멈춤 바로 뒤에 작성한 글자/수식이 무엇인지
   - **해석**: 이 멈춤이 사고/계산/망설임/확인 중 무엇을 의미하는지

2. **멈춤 해석 기준**:
   - **짧은 멈춤 (0.5s~2s)**: 계산 확인, 다음 단계 구상
   - **중간 멈춤 (2s~5s)**: 개념 회상, 전략 수정, 계산 수행
   - **긴 멈춤 (5s 이상)**: 문제 재해석, 풀이 방향 전환, 큰 어려움

3. **사고 흐름 패턴**:
   - 멈춤이 풀이 초반에 집중 → 문제 이해/전략 수립에 시간을 투자한 것
   - 멈춤이 중반에 집중 → 풀이 과정에서 어려움을 겪은 것
   - 멈춤이 거의 없이 빠르게 작성 → 이미 알고 있는 유형이거나, 충분한 사고 없이 작성한 것

4. **필기 속도 변화 분석**:
   - 동영상에서 필기 속도가 빨라지거나 느려지는 구간을 파악하세요.
   - 빠른 필기: 자신감 있는 구간 또는 단순 반복 작업
   - 느린 필기: 신중하게 계산하거나 확인하는 구간

### **[종합 행동 분석]**

위의 풀이 순서와 딜레이 분석을 종합하여, 학생의 **풀이 행동 유형**을 다음 중 하나로 판단하세요:

- **체계적 풀이형**: 문제를 읽고 → 전략을 세우고(초반 멈춤) → 순서대로 풀이 → 답 도출.
- **직관적 풀이형**: 멈춤이 적고 빠르게 작성. 해당 유형에 익숙하거나 직관적으로 접근.
- **시행착오형**: 수정 흔적이 많고, 중간에 긴 멈춤이 있음. 여러 접근을 시도한 흔적.
- **결과 역추적형**: 답을 먼저 쓰고 풀이를 나중에 채움. 풀이 과정의 진정성 의심.
- **포기/미완성형**: 작성 도중 긴 멈춤 후 필기가 중단됨. 완성하지 못한 풀이.

이 행동 유형 판단은 채점 등급을 변경하지 않지만, **피드백의 방향성을 결정하는 중요한 참고 자료**입니다.
